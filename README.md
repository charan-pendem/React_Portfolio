====================================
PROJECT OVERVIEW
====================================

This is a personal portfolio website built using React.js to showcase my projects, skills, and experience as a Full Stack Developer. The portfolio is designed with a clean, responsive layout and focuses on clarity, usability, and professional presentation.

====================================
FEATURES
====================================

- Multi-page portfolio layout
- Styled using React-Bootstrap and custom CSS
- Fully responsive across devices
- Reusable React components
- Projects section with detailed descriptions
- Contact form powered by EmailJS
- Clean and minimal UI suitable for recruiters

====================================
TECH STACK
====================================

Frontend:
- React.js
- React-Bootstrap
- CSS

Routing:
- React Router

Forms & Email:
- EmailJS

Deployment:
- Vercel / Netlify

====================================
GETTING STARTED
====================================

Prerequisites:
- Node.js
- Git

Installation:
1. Clone the repository
   git clone https://github.com/charan-pendem/portfolio.git

2. Navigate to project directory
   cd portfolio

3. Install dependencies
   npm install

4. Start development server
   npm start

The app runs at:
http://localhost:3000
